# puget-salmon-health
Skagit River Chinook Salmon Population Health Dashboard
Overview
This interactive web-based dashboard visualizes the health and status of Chinook salmon populations in the Skagit River watershed of Washington State. The project combines geospatial analysis, historical population data, and environmental monitoring to tell the story of salmon recovery challenges and opportunities in the Pacific Northwest.

Purpose
Chinook salmon in the Skagit River are ESA-listed (threatened), with populations declining significantly over the past 30+ years. This dashboard aims to:
Visualize population trends - Show historical spawning run counts and identify concerning patterns
Map critical habitat - Display spawning and rearing habitat zones relative to current conditions
Identify barriers - Locate fish passage obstacles (dams, culverts) that block salmon access to habitat
Highlight restoration opportunities - Show where conservation projects can have the greatest impact
Communicate environmental context - Connect salmon decline to measurable threats (warming water, habitat loss, dam impacts, ocean conditions)

Features
Interactive Map
Skagit River basin outline - Geographic context and watershed boundaries
Spawning habitat zones - Color-coded by suitability (optimal, marginal, unsuitable)
Water temperature monitoring - Shows seasonal temperature patterns critical for salmon survival
Fish passage barriers - Identifies dams, culverts, and natural barriers blocking upstream migration
Restoration projects - Maps active and proposed habitat improvement initiatives
Clickable popups - Get detailed information about any feature on the map
Population Trends Analysis
Historical spawning counts - 20+ years of WDFW monitoring data
Interactive trend chart - Visualize population trajectory and identify key decline periods
Comparison metrics - Current population vs. recovery targets
Year-over-year changes - Quantify population increase/decrease
Environmental Context
Temperature analysis - Seasonal patterns and impacts on salmon spawning success
Habitat suitability scoring - Based on scientific criteria for Chinook salmon
Threat identification - Visual representation of key limiting factors
Ocean condition indicators - Connect marine survival to freshwater returns (2-year lag)
